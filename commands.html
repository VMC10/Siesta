Commands Mô tả Tham số
help Xem danh sách lệnh -
level Xem level của thành viên - or user
rank Xem rank của thành viên - or user
bank Kiểm tra số coin trong tài khoản -
daily Điểm danh nhận coin hằng ngày -
ck Chuyển coin cho người khác [user][coin]
ban Cấm thành viên khỏi server user or user-id
unban Bỏ cấm thành viên khỏi server user-id
kick Đuổi thành viên khỏi server user or user-id
timeout Đặt thời gian chờ (Cấm trò chuyện và vào kênh thoại) [user][time][reason]
untimeout Bỏ thời gian chờ thành viên [user][reason]
translate Dịch ngôn ngữ khác sang tiếng Việt [words]
ui Xem thông tin của thành viên trong server - or [user]
si Xem thông tin của server -

ht, hh, hn Lấy ngẫu nhiên ảnh (top, hot, new tương ứng với lệnh) về chủ đề hentai trên reddit (18+) -

stats Xem thông tin của bot -
